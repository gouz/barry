Si on se retrouvait au milieu du chemin ?

Cet outil est fonctionnel uniquement en France pour le moment.

Il est basé sur 3 calculs :

  • Le mi-chemin en temps ou en distance sur le plus long trajet entre toutes les villes
  • La moyenne des coordonnées des villes
    • Le centre du cadre dans lequel toutes les villes sont présentes

Ainsi on arrive à la moyenne de ces trois points pour obtenir le pointqui correspond à une approximation du point le plus équitable pour les trajets de chacun.
